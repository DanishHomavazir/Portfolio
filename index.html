HTML 5

Web Development

HTML5 is the latest evolution of the standard markup language used to structure and present content on the World Wide Web. It is the fifth revision of the HTML standard, hence the name HTML5.It introduces new attributes that provide more meaningful semantics to web content.

CSS

Web Development

CSS, or Cascading Style Sheets, is a technology used for controlling the visual presentation of web pages.It works hand in hand with HTML to define the layout, colors, fonts and other styles of a web page ensuring a consistent and attractive appearance across different devices and browsers.

JavaScript

Web Development

JavaScript is a versatile programming language commonly used in web development to add interactivity and dynamic behavior to websites. It is one of the core technologies of the modern web alongside HTML and CSS JavaScript's versatility,make it an indispensable tool for web developers .

SQL

Database

SQL, or Structured Query Language, is a programming language specifically designed for managing and manipulating relational databases. It serves as a standard for interacting with databases, allowing users to perform a wide range of operations,from creating to modifying data stored within them.
